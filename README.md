# USER MANAGEMENT

# SETUP
1 Create a .env file at the root of the project

2 Add the following variables with you own params

    PORT=3000
    LOG_LEVEL='silly'
    MONGODB_URI=[YOUR MONGODBURL]
    JWT_SECRET=[YOUR SECRET STRING]

3 Run: npm install

4 Run: npm start

5 If you want to run the test, Run: npm test
